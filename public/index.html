RGB Of Life

This page demonstrates Conway's Game of Life in what can be considered 24 dimensions. There are actually 24 simultaneous "games of life" being played on the above board, one for each bit in a 24 bit RGB color value that is the background of the square. However, since we naturally think of white as the "dead" color and black as the "alive" color, the bits have been flipped in this demonstration so that 0 represents alive and 1 represents dead. Thus #fff, which is white, represents a cell that is completely dead, even though all of the bits are "on".

Feel free to browse the Javascript source of this demonstration. It makes heavy use of the jQuery Javascript library. The source code of this demonstration is available on Github.
